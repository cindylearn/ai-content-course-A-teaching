# AI 内容营销系统搭建器（任何行业都能用）

**装上这个，Claude 就会主动带你，一步步搭出「你自己行业」的一站式 AI 内容营销系统 / Agent。**

> 不管你是卖课、开服装店、做美容院、搞补习、做 SaaS、还是任何生意——
> Claude 会先问你的行业/品牌，然后按里面的框架，陪你把 **文案 · 图像 · 数字人 · PPT · 音乐 · 影片** 六条 AI 生产线，串成一套能自己跑、能放量的内容系统。
>
> 📌 里面的 **SUMA College** 只是**示范例子**（让你看到"填满后长什么样"）。用的时候，Claude 会换成**你的**行业。

---

## 🚀 3 步开始

1. **下载**：右上角绿色 **Code → Download ZIP**，解压后整个文件夹放进你的 Claude skills 目录。
2. **注册账号**（一次过）：Claude / Notion / Higgsfield / Suno / GitHub —— 见 `03-SETUP`。
3. **发一句**，Claude 就主动开始带你：
   ```
   用 ai-marketing-agent-system skill 带我搭我自己的 AI 营销内容系统。
   ```
   → 之后你只需要**答它的问题 + 看成品挑刺**，不用背 prompt。
